## Commands Used
[your commands]
- touch = to create an empty file
- echo "content" > file_name = add content to file_name (>> if used this then it will append contents to any file)
- vi file_name = to create file in edit mode
- chmod = changing permissions
- cat "file_name" = to view the file contents
- head "file_name" = to print the starting of any file (-n/-20 can be used to see number of lines here it will show 20 top lines)
- tail "file_name" = it will print last lines of any file (-20 will show last 20 lines of file)
- ls -la = list all file includes hidden files in the directory with permissions, user - group permissions, file size, date modified, file name
